## 🚀 Deployment Guide

### 1. Prerequisites
- Node.js 18+
- PostgreSQL instance
- Docker (for production parity)

### 2. Installation & Orbit
```bash
git clone https://github.com/iamdakshsainii/devops-hub.git
cd devops-hub
npm install
```

### 3. Environment Ignition
Create a `.env` file in the root:
```env
DATABASE_URL="postgresql://user:password@localhost:5432/devops_db"
NEXTAUTH_SECRET="vetted-mastery-secret"
NEXTAUTH_URL="http://localhost:3000"
```

### 4. Database Fusion
```bash
npx prisma generate
npx prisma migrate dev
npx prisma db seed # Populate roadmaps & modules
```

### 5. Launch
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) to enter the cockpit.
